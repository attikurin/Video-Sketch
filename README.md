Video Sketch - 動画＆カメラ お絵かきツール

動画ファイル、画像、またはWebカメラの映像の上に、ブラウザ上でリアルタイムに手書きのメモやイラストを書き込めるWebアプリケーションです。

HTMLファイル1つで構成されており、特別なソフトのインストールは不要です。

主な機能

🖊️ リアルタイム描画

動画を再生しながら、または一時停止中に自由に書き込みができます。

ペンの色、太さの変更、消しゴム機能に対応。

📂 マルチメディア対応

動画読込: MP4などの動画ファイルを読み込んで再生。

画像読込: 写真や図解画像を背景として読み込み。

カメラ: PCやタブレットのカメラ映像をリアルタイム表示。

💾 保存・録画

画像保存: 現在の画面（背景＋書き込み）をPNG画像としてダウンロード。

録画機能: 操作の様子（動画再生＋書き込み過程）を動画ファイル（WebM/MP4）として録画・保存。

📱 レスポンシブ対応

PCのマウス操作だけでなく、iPadなどのタブレットやスマホでのタッチ操作にも対応。

使い方

起動: video_draw_app.html をブラウザ（Google Chrome, Edge, Safariなど）で開きます。

メディア選択: 画面右上のボタンからソースを選びます。

動画読込: ローカルの動画ファイルを選択。

画像読込: ローカルの画像ファイルを選択。

カメラ: Webカメラを起動（許可を求められた場合は「許可」してください）。

描画: 画面上をドラッグ（またはタッチ）して描画します。

画面下部のコントロールバーで、再生/一時停止、色、ペンの太さ、消しゴムを切り替えられます。

保存:

画像保存（紫のボタン）: 現在の瞬間を画像として保存します。

録画（録画ボタン）: 押すと録画開始（赤く点滅）、もう一度押すと録画を停止し、動画ファイルをダウンロードします。

動作環境

推奨ブラウザ: Google Chrome (最新版), Microsoft Edge, Safari

ネットワーク: 起動時にスタイルシートとアイコンライブラリ（CDN）を読み込むため、インターネット接続が必要です。

※完全にオフラインで使用したい場合は、HTML内のCDNリンク（Tailwind CSS, Lucide Icons）をローカルファイルへのパスに書き換える必要があります。
